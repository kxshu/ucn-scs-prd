# 合同与订单

### 定义

`采购商`和`供应商`采购需求达成一致后签署合同，并创建采购订单。

### 角色

| 角色 | 参与状态 |
| :--- | :--- |
| 采购商 | Y |
| 供应商 | Y |
| 服务商 | N |

### 术语

无

### 流程

![](/assets/合同及订单.png)

### 合同方式

1. EXW（需要`FOB`服务时，`供应商`具备`FOB`能力） 双份合同，与`服务商`签合同，且双份合同里价格不一样。
2. FOB（不需要`FOB`服务时，`供应商`不具备`FOB`能力） 双份合同，采购商与供应商签合同，且双份合同里价格一致。
